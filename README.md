# tokengrab-automod
Ths program is a Discord Bot. It will check every incoming message within all channels and delete all "bro" messages containing pics, often sent by tokengrab bots. Since these bots follow the same pattern, the messages can be easily purged.
The bot does not ban the tokengrabbed user, nor can it prevent any contact within DM, but it can at least prevent visual and message pollution.

# How to use

- Install Python and discord.py library
- Fill your app token within TOKEN variable
- Run `python tokengrab-automod.py`
- If you want to keep control of the console, add `&` at the end of the command to run bot on background
- If you want the bot to persist even after logging out of terminal, use this command instead : `nohup python tokengrab-automod.py &` 
- If you want to save logs, add `>> tokengrab-automod.log` at the end of the command to save all output to tokengrab-automod.log in the current folder.

# Scope

This bot has the following Permissions :
- Manage messages
- See channels
